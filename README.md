# 📘 Miniguia de Estudo

## 1. A Ciência do Aprendizado

Machine Learning (ML) é uma área da Inteligência Artificial que permite aos computadores aprenderem padrões a partir de dados sem a necessidade de programação explícita para cada tarefa.

### Principais Categorias

### Supervisionado

Utiliza dados rotulados para aprender relações entre entradas e saídas.

Exemplos:

* Previsão de preços;
* Detecção de fraudes;
* Diagnóstico médico.

### Não Supervisionado

Busca padrões ocultos em dados sem rótulos.

Exemplos:

* Segmentação de clientes;
* Agrupamento de documentos;
* Sistemas de recomendação.

### Aprendizado por Reforço

O agente aprende por tentativa e erro através de recompensas e penalidades.

Exemplos:

* Jogos;
* Robótica;
* Veículos autônomos.

---

## 2. A Revolução dos Transformers

Os modelos modernos de IA generativa, como o ChatGPT, utilizam a arquitetura Transformer.

Seu principal diferencial é o mecanismo de Atenção (Attention Mechanism), que permite analisar simultaneamente todas as palavras de uma sequência e compreender melhor o contexto.

### Benefícios

* Melhor compreensão semântica;
* Maior capacidade de processamento paralelo;
* Melhor desempenho em tarefas de linguagem natural;
* Escalabilidade para grandes volumes de dados.

---

## 3. Limites da Inteligência Artificial

Apesar dos avanços recentes, existem limitações importantes.

### Problema da Parada

Proposto por Alan Turing, demonstra que não existe um algoritmo capaz de determinar corretamente, em todos os casos, se outro algoritmo irá terminar sua execução.

### Quarto Chinês

Experimento mental proposto por John Searle.

A crítica sugere que manipular símbolos seguindo regras não significa necessariamente compreender seus significados.

Isso levanta discussões sobre consciência, entendimento e inteligência genuína.

| NLP              | Conjunto de técnicas que permite às máquinas compreender e gerar linguagem humana.                |
| Word Embeddings  | Representação vetorial de palavras para processamento matemático de significados.                 |
| Tokens           | Fragmentos de texto utilizados pelos modelos de linguagem.                                        |
| Transformers     | Arquitetura de redes neurais baseada em mecanismos de atenção.                                    |
